# **Seat-allotment**
Allots seat according to student preference on basis of rank and availability
<br>
Enter the number of students and choices they can choose( they can put -1 if they dont want to exhaust their choices)
<br> 
Also enter their rank simultaneously 
<br>
Then add details of seat matrix like number of colleges and number of seats 
<br>
Then enter students preference for college ( NOTE: 1st PREFERENCE WILL BE CHOSEN FIRST IF AVAILABLE ELSE GO TO NEXT CHOICE)
<br>
If no seat alloted to student then it will show "null"
<br>
**CURRENTLY WORKING ON ZERO SEAT COLLEGE EXCLUSION AND THE COLLEGE PREFERENCE ENTERED MUST BE IN THE GIVEN LIST OF SEAT MATRIX**
